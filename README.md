# ECON 426: Homework 08 - Panel Data Methods

## Assignment Overview
Complete the assignment by following the instructions in the Quarto document.

## Files
- `Homework08.qmd` - Quarto document with assignment instructions
- `_quarto.yml` - Quarto project configuration

## Instructions
1. Open this project in RStudio
2. Open `Homework08.qmd`
3. Complete the assignment by following the instructions
4. Render to MS Word by clicking "Render" or running `quarto render Homework08.qmd`
5. Submit the generated `.docx` file to Canvas

## Rendering
To render the document to MS Word:
```r
quarto::quarto_render("Homework08.qmd")
```

Or use the Render button in RStudio.
